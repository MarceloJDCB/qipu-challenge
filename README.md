
## Comandos Make

- Rebuildar os containers sem cache
    ```
    _rebuild
    ```

- Rebuildar os containers com cache
    ```
    rebuild
    ```

- Iniciar os containers
    ```
    up
    ```

- Rodar o desafio 1
    ```
    lista_encadeada
    ```

- Rodar o desafio 2
    ```
    aisweb
    ```

- Buildar os containers
    ```
    build
    ```

- Lista logs
    ```
    logs
    ```

- Remove containers
    ```
    down
    ```

- Para todos os containers
    ```
    stopall
    ```

- Inicia um .env
    ```
    _cp_env_file
    ```
